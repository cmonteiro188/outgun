# Outgun

Historical preservation repository of the source code formerly maintained by Niko Ritari ("Nix") and Jani Rivinoja ("Huntta") for the game Outgun.

## Background

Following the disappearance of the original infrastructure previously hosted under mbnet.fi, an effort was made to recover and preserve the latest available material related to Outgun.

In August 2025, Nix provided Git bundle backups containing the repositories available to him, including their commit history and development branches.

This repository is a direct import of the source code bundle received from Nix, preserved in its original form.

## Preservation Tag

The tag

`nix-recovery-2025`

marks the repository state exactly as recovered from the original bundle, before any subsequent modifications.

## Status

This repository is preserved primarily for historical and archival purposes.

The contents reflect the state of the source code as recovered from the original backups.
