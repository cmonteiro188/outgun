    OUTGUN

 A 2D-graphics, 32-player multiplayer (low net usage), fast-paced capture-the-flag game!

-------------------------
NEW in version 0.5.0
-------------------------

- added new option for dedicated servers: "-text" for machines without a GUI. example:
     outgun.exe  -ded  -text      (good for UNIX or GNU/Linux servers not running X!!!)
- fixed small bug in netcode (dropped flags appearing in weird places)

-------------------------
NEW in version 0.4.9
-------------------------

- SERVER-SIDE ONLY UPDATE: fixed bug in server wich caused lost ranking score updates

-------------------------
NEW in version 0.4.8 BETA
-------------------------

- many small bug fixes
- all-new ranking stuff
- hit F3 to show/hide player names drawn on top of the characters!

-------------------------
NEW in version 0.4.7
-------------------------

- player talk now has an "anti-flooding" filter
- player password saved to PASSWORD.BIN, so you don't have to type it over and over again
- servers that have a "private IP" (10.x.x.x or 192.168.x.x) and other "regular" IP, now
  publish the right IP address to the master server
- one possible cheat with the Outgun CTF Ranking was fixed
- many other small bug fixes
